# 0.1.1 — 2026-05-23

## Added

- **`socrata_find_datasets`** — cross-portal dataset discovery via the Discovery API; full-text search, domain scoping, category/tag/asset-type filters, sort options, pagination
- **`socrata_get_dataset`** — dataset schema and metadata inspection; typed column definitions with Socrata data types, row count, licensing, and WHERE-clause quoting guidance
- **`socrata_query_dataset`** — SoQL query execution with structured params (select, where, group, having, order, search, limit, offset); echoes assembled SoQL string; DataCanvas spillover when `CANVAS_PROVIDER_TYPE=duckdb`
- **`socrata_list_portals`** — browse the known Socrata portal catalog; client-side substring filtering on domain or organization name, pagination up to 200 per page
- **`socrata_dataframe_describe`** — list DataCanvas tables registered in the current session with DuckDB-inferred column types and row counts
- **`socrata_dataframe_query`** — run SELECT-only SQL against DataCanvas tables; DuckDB type inference enables numeric comparisons on SODA string columns
- **`socrata://datasets/{domain}/{datasetId}`** resource — stable URI for dataset metadata; same payload as `socrata_get_dataset`
- **`socrata://portals`** resource — paginated portal catalog via stable URI
- **`explore_open_data`** prompt — structured six-step civic data investigation workflow
- MCPB bundle support via `manifest.json` for one-click Claude Desktop install
- Dockerfile with OCI labels; STDIO and Streamable HTTP transports
- Built on `@cyanheads/mcp-ts-core` ^0.9.7
