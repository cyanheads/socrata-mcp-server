# 0.1.0 — 2026-05-23

## Added

- **`socrata_list_portals`** — list known Socrata-powered government open-data portals with domain, organization, and dataset count; client-side substring filtering and pagination
- **`socrata_find_datasets`** — full-text search across all Socrata portals via the Discovery API; domain scoping, category/tag filters, asset type filtering, sort options, pagination
- **`socrata_get_dataset`** — fetch full metadata and typed column schema for a dataset by ID; columns include field name, Socrata data type, description, and non-null count
- **`socrata_query_dataset`** — execute SoQL queries with select, where, group, having, order, search, limit, offset; assembles and echoes the final SoQL string; DataCanvas spillover when `CANVAS_PROVIDER_TYPE=duckdb`
- **`socrata_dataframe_describe`** — list registered tables in a DataCanvas session with DuckDB-inferred column types and row counts
- **`socrata_dataframe_query`** — run SELECT-only SQL against DataCanvas tables; DuckDB type inference enables numeric comparisons on SODA string columns
- **`socrata://datasets/{domain}/{datasetId}`** resource — stable URI for dataset metadata, same payload as `socrata_get_dataset`
- **`socrata://portals`** resource — paginated portal catalog via stable URI
- **`explore_open_data`** prompt — structured six-step civic data investigation workflow
- **`SOCRATA_APP_TOKEN`** env var — optional app token for higher per-IP rate limits
- **`SOCRATA_DEFAULT_DOMAIN`** env var — configurable fallback portal domain (default: `data.seattle.gov`)
- Dockerfile with OCI labels; STDIO and Streamable HTTP transports
- MCPB bundle support via `manifest.json` for one-click Claude Desktop install
